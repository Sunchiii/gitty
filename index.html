Why Teams Love Gitty

Branch Protection

Prevents direct commits to protected branches (main, uat, develop) with automated checks and pre-push hooks.

Workflow Enforcement

Enforces Git flow-like policies with consistent branch naming and proper merge strategies.

Auto Sync & Rebase

Automatically pulls latest changes and handles rebasing to prevent merge conflicts and keep history clean.

Conflict Prevention

Detects potential merge conflicts early and provides clear guidance to resolve issues before they become problems.

Branch Cleanup

Automatically cleans up merged branches and keeps your repository organized with minimal manual intervention.

Team Consistency

Ensures all team members follow the same workflow patterns with helpful error messages and guidance.

Simple Workflow

1
Start a new feature
gitty start feature new-awesome-feature

Automatically switches to develop, pulls latest changes, and creates your feature branch.

2
Make changes and commit
git add . && git commit -m "Add awesome feature"

Work normally with standard Git commands. Gitty protects you in the background.

3
Stay in sync
gitty sync

Fetches latest changes and rebases your branch with develop to prevent conflicts.

4
Push and create PR
git push origin feature/new-awesome-feature

Push your changes safely. Gitty prevents pushes to protected branches.

5
Finish the feature
gitty finish feature new-awesome-feature

Merges to develop with --no-ff flag and cleans up the feature branch.

Powerful Commands

gitty start
Create new feature or hotfix branches with proper base branch setup and latest changes.
gitty finish
Complete features and hotfixes with proper merging and automatic branch cleanup.
gitty sync
Keep your branch up-to-date with develop through smart rebasing and conflict detection.
gitty check
Validate branch rules and status before pushing to prevent common mistakes.
gitty protect
Install Git hooks to automatically protect critical branches from direct pushes.
gitty conflict
Detect potential merge conflicts early to resolve issues before they become problems.
gitty cleanup
Remove merged branches and sync with remote to keep repository organized.
gitty update
Update Gitty to the latest version with automatic backup and platform detection.
